# Simple-Sig
Simple Signature to secure your website's traffic with backend validation api

#### Usage example
```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>sig demo</title>
  </head>
  <body>
	<script src="https://cdn.jsdelivr.net/gh/xtekky/simple-sig/sig.js"></script>
  </body>
</html>
```
